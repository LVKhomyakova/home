<nav> Навигация: внутри список <ul> из 6 пунктов <li>. Где пункт  - текст-ссылка <a>.

<header> Шапка: внутри 2 контейнера:
 - <img> картинка для лого + <h> заголовол для названия организации.
 - блок поиска (элемент ввода).

<main> Главный контент:
 1 - <div> карусель картинок: <img> + подпись <figure>.
 2 - <article> внутри 4 секции <section>. Где каждая секция содержит фоновое изображение <img> + заголовок <h> + текст <p> + кнопка <button>
 3 - <article>  внутри две колонки:
  3.1 - <section> внутри:
    3.1.1 - заголовок <h>.
    3.1.2 - <section> секция с контентом: заголовок <h> + дата <time> + картинка <img> + текст <p>
    3.1.3 - <button> кнопка.
  3.2 - <sidebar> внутри:
    3.2.1 - <h> заголовок
    3.2.2 - <section> секция со списком <ul> новостей. Где пункт списка это маркер <time> + текст <p>.
    3.2.3 - <button> кнопка.
 4 - <article> внутри заголовок <h>

<footer> внутри:
 1 - <article> c двумя <section>. В секции текст <p>.
 2 - <nav>: список <ul> контактов: маркер <img> + текст <p>
 3 - <article> список <ul> элементов для обратной связи + кнопка <button>
 4 - <nav> авторское право, политика.
